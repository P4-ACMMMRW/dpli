## How to Compile
On Ubuntu simply run:
```bash
./scripts/compile.sh
```

On other linux distrobutions:
```bash
mkdir build
cd build
cmake ..; make
```

### Dependencies
- `Java JDK 11+`
- `cmake`
- `make`
